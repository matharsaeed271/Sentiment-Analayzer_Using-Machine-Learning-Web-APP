# Sentiment-Analayzer_Using-Machine-Learning-Web-APP

####  Now scale and classify your thoughts with the cutting edge web app
This web application is designed to analyze the sentiment of a given text using machine learning techniques. It classifies the input as positive, negative, or neutral based on the emotions expressed in the text.

The application provides a simple and user-friendly interface where users can enter text (reviews, comments, or messages) and instantly receive sentiment results.

This project demonstrates the real-world use of machine learning in opinion mining, customer feedback analysis, and social media monitoring.

The web app is available online and can be accessed here:

👉 https://sentitrack.streamlit.app/
